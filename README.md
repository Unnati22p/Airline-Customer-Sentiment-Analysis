Airline Customer Sentiment Analysis

A Natural Language Processing (NLP) project that analyzes airline customer tweets and classifies customer sentiment as positive, neutral, or negative.

Tools & Technologies

* Python
* Pandas
* NumPy
* NLTK
* Scikit-Learn
* Matplotlib
* Seaborn
* WordCloud

Dataset

The project uses airline customer tweets containing passenger feedback and service experiences.

Analysis Performed

* Text preprocessing and cleaning
* Sentiment distribution analysis
* Airline-wise sentiment comparison
* TF-IDF feature extraction
* Logistic Regression model training
* Classification report evaluation
* Confusion matrix visualization
* Word cloud generation

Files

* airline_sentiment_analysis.ipynb
* Tweets.csv
